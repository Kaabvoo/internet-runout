# Internet Outage

## Description

Made using [Nest](https://github.com/nestjs/nest)

The purpose of this repo is to have a palce to check when a power outage or internet outage happen at home.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev
```
